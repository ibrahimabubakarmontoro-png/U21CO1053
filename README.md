# 📇 Business Card App

A modern Android Business Card application built using **Kotlin** and **Jetpack Compose**.

## 🚀 Features

- Circular profile image
- Material 3 design
- Clean Compose UI layout
- Responsive spacing
- Contact information section

## 🛠 Built With

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio



## 📦 Project Structure

- `MainActivity.kt` → App entry point
- `BusinessCard()` → Main UI composable
- `ContactRow()` → Reusable contact component
